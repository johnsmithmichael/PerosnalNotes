 1.if语句
	1.1if的第一种格式
		if(boolean表达式){
		执行语句；;
		}
	1.2if的第二种格式
		先走if后面的boolean表达式 ，如果是true，就走if后面的大括号中的表达式，否则就不走
			1.2.1注意：
			if后面跟的是boolean表达式，只要结果为boolean型的，都可以放在小括号中，哪怕直接写一个true或false
	注意 ”=“是赋值而” == “是判断等于否这里是要注意的，例如赋值一个true，那么他就只能顺序下去不会否则
